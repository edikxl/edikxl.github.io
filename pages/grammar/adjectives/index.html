Adjectives
い-adjectives
形容詞「けいよおし」
All い-adjectives end in the vowel /i/. This is no coincidence. This is because the /i/ is an ending that attaches to the stem. The stem of a word is its root, what makes up the meaning of the word. The い is a grammatical item that adds a finishing touch. The resultant form is the plain non-past form of an adjective. Although conjugation is technically involved, speakers treat the "stem" and the ending い as parts of a single word. Knowing why the い is there simply helps you know when something is a い-adjectives

な-adjectives
形容動詞「けいよおどおし」
This name 形容動詞 literally means "adjectival verb". The name comes from the fact that all adjectives in this class must use the copula to conjugate. Confusingly, however, they’re essentially adjectival nouns.
The な-adjective is very simple to learn because it acts essentially like a noun. Grammatically, an abstract noun is used with a form of the copula. Remember, the copula simply refers to either だ, です, or any of their conjugations. Although this means the conjugations will be exactly the same as the copula
な-adjective can directly modify a noun following it by sticking な between the adjective and noun. For example:
魚さかなfish が 好すきlike な 人ひとperson。
Person that likes fish.
This type of sentence is useful because 「人は魚が好きだ」 would mean "People like fish", which isn’t always the case.
We can even treat the whole descriptive noun clause as we would a single noun. For instance, we can make the whole clause a topic like the following example:
魚さかなfish が 好すきじゃないlike(negative) 人ひとperson は 、 肉にくmeat が 好すきlike だ 。
Person who does not like fish like meat.
Not all adjectives that end in /i/ are い-adjectives. If the /i/ part is written in 漢字かんじKanji, or can be written in 漢字かんじKanji, it will always be a な-adjective. Even if the final /i/ is written in かな, if it can be followed by だ, it's a な-adjective.
Exceptions
There is one i-adjective meaning "good" that acts slightly differently from all other i-adjectives. The word for "good" was originally 「よい（良い）」. However, with time, it soon became 「いい」. When it is written in Kanji, it is usually read as 「よい」 so 「いい」 is almost always Hiragana. That’s all fine and good. Unfortunately, all the conjugations are still derived from 「よい」 and not 「いい」.
Another adjective that acts like this is 「かっこいい」good-lookingbecause it is an abbreviated version of two words merged together: 「格好」かっこうlooks and 「いい」good. Since it uses the same 「いい」, you need to use the same conjugations.
Take care to make all the conjugations from 「よい」 not 「いい」
